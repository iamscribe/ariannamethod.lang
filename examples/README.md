# Example DSL Scripts for ariannamethod.lang

This directory contains example DSL command sequences demonstrating various field configurations.

## Files

- **prophecy_basic.dsl** — Basic prophecy mechanics
- **wormhole_travel.dsl** — Calendar wormholes and time travel
- **velocity_modes.dsl** — Velocity operator demonstrations
- **codes_ric.dsl** — CODES/RIC structured resonance
- **dark_matter.dsl** — Dark matter learning and scars
- **entity_control.dsl** — Entity intention systems
- **field_sculpting.dsl** — Advanced field manipulation

## How to Use

1. Open ariannamethod.lang in browser
2. Copy commands from a .dsl file
3. Paste into the input field (bottom of screen)
4. Press Enter to execute
5. Observe geometry changes (walls, entities, metrics)

## Note on Input Modes

- **UPPERCASE** = DSL command (executes immediately)
- **lowercase** = text injection (deforms field through movement)

Commands in these examples are UPPERCASE for DSL execution.
